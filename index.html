This app shows the different legs that make up the Ring of Fire Ultra-Marathon.

Disclaimer

This is a non-commercial, demo application built upon the hiking app made by Nicola Raluca at the Esri R&D Center Zurich for visualizing the Ring of Fire Ultra Marathon. It is built on top of ArcGIS API for JavaScript version 4.10. This app has been built independently of the Ring of Fire as a demonstration and is not endorsed or affiliated officially with the event.

Credits

Data for the trails was sourced from GPS tracks taken during the 2018 Ring of Fire.
